# Pac_Man-Game
Basic Level Java Pacman Game 

## Description
This is a simple Pacman game made in Java.
It includes:
* Pacman (P)
* Ghost (G)
* Food system

## How to Run
1. Open project in IntelliJ
2. Run the MainGame file
3. Follow on-screen instructions

## Features
* Grid-based movement
* Random ghost movement
* Score system
