# 🎮 Pacman Game (Java)
## 📌 Overview
This project is a simple console-based Pacman game developed in Java.
The game features a grid system where the player controls Pacman (P) while avoiding a ghost (G) and collecting food.
---

## 🧠 Game Concept
* Pacman moves inside a 2D grid
* The ghost moves randomly
* Food appears on the board
* The goal is to collect food while avoiding the ghost
---

## ✨ Features
* ✔ Grid-based game board
* ✔ Player-controlled Pacman
* ✔ Random ghost movement
* ✔ Food collection system
* ✔ Score tracking 
---

## 🛠️ Technologies Used
* Java
* IntelliJ IDEA
---

## ▶️ How to Run the Game
1. Open the project in IntelliJ IDEA
2. Locate the main file (`MainGame.java`)
3. Run the program
4. Use keyboard input (as defined in code) to move Pacman
---

## 🎮 Controls
* W → Move Up
* S → Move Down
* A → Move Left
* D → Move Right

---

## 📂 Project Structure
* `MainGame.java` → Main game logic
* `Food.java` → Handles food placement
* Other files → Supporting classes
